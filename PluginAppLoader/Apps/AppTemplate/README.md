RoboDK App Template / Example
=================================

This App provides the necessary examples to build a RoboDK App.
It can be used as a template to build your own RoboDK Apps.

It includes:

* Actions
    * Checkable
    * Momentary
    * Contextual
    * Double-click (for generic items)

* Options
    * Individual
    * Grouped

* Settings
    * GUI settings

* Icons
